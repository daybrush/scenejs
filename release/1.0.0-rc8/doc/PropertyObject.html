Name Type Description

index? Number The index of the current element being processed in the array.

Example
//rgba(100, 100, 100, 0.5)
  var obj4 = new PropertyObject([100,100,100,0.5], {
      "separator" : ",",
      "prefix" : "rgba(",
      "suffix" : ")"
  });

  obj4.forEach(t => {

  });  // =>   "100,100,100,0.5"
